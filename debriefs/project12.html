Phronesis training debrief (N12)

  • What made you smile?
    • Pretty well documentation. Has all necessary examples to complete.
  • What did you find confusing? Did you encounter any issues?
    • It's undoable for people without a tech background. Documentation can only be used with a server that you can access by ssh or can deploy it with a startup command(to run the web server). It couldn't be completed with using a free Render service version because it can run only static websites. It could be completed with netlify, but to do this, you need to figure out a Netlify project structure and create a proper Netlify config, which also could be a challenge.
